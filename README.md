# eyescrape

    scrape every news

## Installation

`go get -u github.com/knowbee/eyescrape`

## Usage

```
  eyescrape <options>
```

## Options

    igihe: Fetch headlines from igihe.com

    inyarwanda: Fetch healines from inyarwanda.com

    thechronicles:  Fetch headlines from thechronicles.rw
