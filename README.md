## Installation

`go get -u github.com/knowbee/eyescrape`

## Usage

```
  eyescrape <options>
```

## Options

    igihe: Fetch news headlines from igihe.com

    inyarwanda: Fetch news healines from inyarwanda.com

    thechronicles:  Fetch news headlines from thechronicles.rw
